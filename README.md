# Quiz Interativo em JavaScript

Um quiz simples e interativo feito com **HTML, CSS e JavaScript**.  
Você pode responder perguntas e ver seu desempenho no final!

## Funcionalidades

- Perguntas de múltipla escolha
- Feedback instantâneo de resposta certa ou errada
- Mostra a pontuação ao final
- Botão para **reiniciar o quiz**
- Animações e efeitos visuais

## Tecnologias usadas

- HTML5
- CSS3
- JavaScript (ES6)
- Git e GitHub Pages para deploy
